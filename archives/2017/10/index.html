行为检测论文笔记：Temporal Segment Networks

Jelon 发表于 2017-10-24    标签： Deep Learning / 深度学习 / Action Detection / 行为检测 / Action Recognition / 行为识别

Abstract

  • 本文旨在发现设计有效的ConvNet体系结构以实现视频中的动作识别的原则，并通过有限的训练样本来学习这些模型。
  • 贡献
    • 第一个贡献：TSN网络，一个新的基于视频的动作识别框架，其基于远程时间结构建模的思想，结合了稀疏的时间采样策略和视频级别（Video-level）监督，以使用整个动作视频实现有效地学习。
    • 第二个贡献：在时间段网络的帮助下，研究一系列的视频数据ConvNets的良好实践。本文的方法在HMDB51（69.4％）和UCF101（94.2％）的数据集上获得了最先进的表现。
    • 本文观察学习到的ConvNet模型，从而定性地展示了时间段网络的有效性并提出了良好实践。
